# Teste-Soema HELP

## Pré requisitos
* Node ou Yarn
* React Native
* Bibliotecas React Native

### Instalando bibliotecas
Use o comando:
```
npm install
```
ou
```
yarn install
```
caso esteja usando yarn.
Isso fará com que todas as dependências necessárias.

## Rodando
Para rodar o aplicativo é necessário você ter seu dispositivo mobile no modo dev, após isso vá nas **Configurações** do seu celular, vá em **Sistema**, **Opções de Desenvolvedor** e habilite a opção **Depuração USB**.

Após isso, conecte seu celular a um computador com um cabo USB. Certifique-se de que esse cabo permita a passagem de dados.

Após isso, no terminal, digite:
```
npm run android
```
caso seu aparelho seja um android
```
npm run ios
```
caso ele seja um aparelho apple.

Após isso digite:
```
npm start
```
e no local onde  é indicado para adicionar a URL do aplicativo, digite a URL "http://localhost:8081". Após isso o aplicativo ira estar rodando em seu dispositivo móvel
